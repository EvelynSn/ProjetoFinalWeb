## Tecnologias Utilizadas

* Frontend: Vue.js 3
* Backend: Node.js + Express
* Banco de Dados: SQLite

---
## Instalação

### Pré-requisitos

* Node.js (versão 16 ou superior)
* npm

### Backend

```bash
cd backend
npm install
npm run start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Execução

* Backend: `http://localhost:3000`
* Frontend: `http://localhost:5173`

O banco de dados SQLite é criado automaticamente ao iniciar o backend.

---

## Páginas do Sistema

* **Página Inicial**: acesso às funcionalidades do sistema
* **Alunos**: cadastro, listagem, edição e remoção
* **Professores**: cadastro e visualização
* **Disciplinas**: cadastro e associação com professores
* **Atividades**: cadastro e acompanhamento de atividades acadêmicas
